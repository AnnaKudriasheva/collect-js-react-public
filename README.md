# collect-js-react
React wrapper for a CollectJS library

## How to run locally 
```bash
git clone
cd ./collect-js-react
yarn install
yarn start

cd ./example
yarn start
open http://localhost:3000
```

## Install

```bash
npm install --save collect-js-react
```
